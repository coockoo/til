# Vim Abbreviations

`iabbrev` command allows adding abbreviations and not only in insert mode.
Useful for typos

```vim
" Insert email
:iabbrev @@ admin@mail.com
" Fix typo
:iabbrev adn and
" Fix typo
:iabbrev cosnt const
```
